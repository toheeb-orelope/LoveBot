**Manage virtual environment using anaconda**

Empty Environment: To create a simple environment by name: conda create --name myenv

With Specific Python Version: (Recommended) Most projects require a specific version: conda 
create -n myenv python=3.14.3

With Initial Packages: You can install packages at the same time you create the environment: 
conda create -n myenv python=3.11 numpy pandas matplotlib

Once created, you must activate it to use it: 
Activate: conda activate myenv
Deactivate: conda deactivate
List all environments: conda env list or conda info --envs
Remove environment: conda env remove -n myenv

Clone an existing environment: Useful for making backups before risky changes: 
conda create --name new_env --clone old_env
